# Quickstart Guide for Project Management System

## Prerequisites

- Docker
- Docker Compose

## Getting Started

1.  **Clone the repository**

    ```bash
    git clone <repository-url>
    ```

2.  **Start the application**

    ```bash
    docker-compose up -d
    ```

3.  **Access the application**

    -   **API**: `http://localhost:8000`
    -   **Frontend**: `http://localhost:3000`
